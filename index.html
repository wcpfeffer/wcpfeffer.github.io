VoIP & Instant Messenger

Technologies: Java, JavaFX, CSS

This is a personal project that started from a general interest in developing an instant messaging system when I first started programming. This project has gone through many iteration, changing languages a few times. It started as a C# client, and was later ported to Java to support a wider variety of systems.

This is an audio and text based communication system. This project uses a client/server architecture, and gave me experience with networking, multi-threading, and porting from C# to Java.

Implemented:

  • JavaFX User Interface
  • Peer to Peer calling (via UDP hole-punching)
  • Encryption
  • Instant Messaging
  • MySQL Database

Co-Dev: Nathan Duncan

Data Provenance Reconstruction

Technologies: Java

This tool aims to reconstruct data provenance records through semantic analysis of documents. It focuses on news article provenance links and provenance links between files in git repositories.

A continuation of work done by graduate students, I improved the reconstruction process by improving the foundational code, and implementing semantic analysis techniques such as cosine similarity.

Developed a GUI and two automation tools to collect datasets to test the reconstruction process.

Book Store

TraceIt

Technologies: Java

TraceIt is an android game that challenges users to trace shapes in a short time window. This was a game developed for my modern software development class, and gave me experience working with mobile development.

I was responsible for the polygon creation and randomization process. I also helped collaborate on various issues and design decisions.

ChangeTrac

Technologies: C#

ChangeTrac is a research tool developed to capture changes made to Microsoft Excel spreadsheets. I was responsible for adding additional features, collecting user feedback, and revising the tool based on the feedback received.

I also implemented a multi-cell tracking technique that allowed tracking changes over multiple cells at the same time, and was responsible for refactoring the entire code base.
